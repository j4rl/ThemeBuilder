# ThemeBuilder

En statisk webbapp för att bygga CSS-teman med harmoniska färgval, typsnitt och stöd för ljus/mörk variant.

## Funktioner

- Välj basfärg, accentfärg och färgharmoni.
- Justera hur tät eller bred färgharmonin ska vara.
- Förhandsvisa tema i ljus och mörk variant.
- Appens eget gränssnitt kan följa systemet eller låsas till ljus/mörk.
- Välj separata Google Fonts för rubriker och brödtext.
- Se om valda typsnitt passar ihop.
- Exportera färger som hex, rgb, hsl, oklch, display-p3 eller alla format samtidigt.
- Exportera färdiga CSS-variabler och enkla basklasser.
- Visa en snabb översikt över basklasserna innan export.
- Ladda ned temat som `.css` eller kopiera CSS direkt.

## Kör lokalt

Öppna `index.html` direkt i webbläsaren, eller kör projektet via en lokal webbserver.
